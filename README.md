# hr-ocr-system
Hệ thống tự động hóa xử lý tài liệu nhân sự, dùng OCR (PaddleOCR) và LLM (LLaMA-3.3 qua Groq API) để trích xuất, phân loại hàng ngàn văn bản. Rules Engine tự chuẩn hóa tên file, lưu trữ qua Google Drive. Giao diện ReactJS hỗ trợ duyệt hàng loạt. Giảm 80% thời gian nhập liệu thủ công.
